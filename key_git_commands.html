Kluczowe komendy programu Git

git config --global user.name ""

Komenda zmieniająca naszą lokalną nazwę w aplikacji Git. Nazwa ta służy do podpisywania commitów wysyłanych do bazy głównej, nie jest w żaden sposób powiązana z naszą nazwą na przykład na GitHubie. Trzeba zaznaczyć, że zmiana nazwy nie spowoduje zmiany w poprzednich komitach, które zostały wysłane na serwer główny, a jedynie wpłynie na każde przyszłe commity. Parametry:

  • --global: ustawia nazwę dla każdego repozytorium, nad którym pracujemy. Bez tego parametru, zmienimy jedynie nazwę w repozytorium, w którym się znajdujemy
  • "" - w cudzysłowiu podajemy swoją nazwę Dodatkowo możemy jeszcze powiązać adres e-mail z gitem zastępując parametr user.name parametrem user.email
